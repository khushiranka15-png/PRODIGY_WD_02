# Stopwatch Web Application

This project is created as part of **Task-02** of my Web Development Internship at **Prodigy InfoTech**.

# Task Description
Create a stopwatch web application that can:
- Start, pause, and reset the stopwatch
- Record lap times
- Display accurate minutes, seconds, and milliseconds
- Provide an interactive and user-friendly interface
